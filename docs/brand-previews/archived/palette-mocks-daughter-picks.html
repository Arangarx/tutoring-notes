Mynk — daughter's picks (batch 2)
14 palettes she pulled · 2026-05-19
For the daughter — round 2

Fourteen palettes she picked out, with hex codes on every swatch.

Two groups: the first 9 are warm/ocean/sunset names from the first screenshot, the last 5 are cool/cozy/muted names from the second. Sunny Beach Day is omitted because it's already in palette-mocks-warm-colors.html.

One honest caveat: these palette names aren't unique to Coolors — names like "Ocean Sunset" and "Cozy Cabin" exist on Coolors, ColorMagic, ColorDrop, iPalettes, and a dozen other sites, each with slightly different hex sets. I picked the best-matching hex set per name based on which version felt most true to the name. The ones marked loose match are where I had to derive a darker text color or where the source palette had fewer than 5 colors. If she points at one and says "that's not the one I saw", hover over that card on Coolors, copy the URL (it'll contain the hex codes), paste it here, and I'll swap it to the exact set instantly.

Toggle light/dark in the header. Each card shows the same 5 surfaces (homepage, dashboard, recording bar, AI field) so she can compare apples to apples.
