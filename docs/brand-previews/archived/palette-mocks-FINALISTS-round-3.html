MWW + MWW v2 + 3 coral placements + M4 v2 (darker) · 2026-05-19

Round 3: MWW with three different coral placements, plus M4 with darker dark mode.

Two threads to evaluate. (1) The daughter wants MWW Light v2 with one more coral accent — but "one more coral accent" is actually three different design questions, so this page shows all three placements side-by-side and you can vote on which one reads best. (2) M4 Forest-Teal Bridge's dark mode needed darker surfaces so the non-bold body text reads cleanly. v2 darkens both surface tones and brightens the muted text.

Cards C, D, E all start from the identical MWW Light v2 token set — the only thing that changes between them is where the coral lives. Read them as "same palette, three jobs for coral":

  • C · Coral as primary CTA — coral takes the accent role globally. Deep forest demotes to a strong-border. Loudest, most "branded warmth."
  • D · Coral as live signal only — only the recording dot is coral. Everything else stays MWW v2 deep-forest. Minimal warmth; coral becomes a status semantic ("we're live with you right now").
  • E · Coral as AI tint — the AI summary panel and the "AI notes" tag use coral; CTAs stay deep forest. Says "AI content = warm/human" while keeping the rest of the system forest.
Methodology note. C, D, E share the exact same MWW Light v2 token set, so any visual difference is purely about coral placement — not about palette differences. Compare the recording bar between D and the others to see the "live signal" effect cleanly; compare the AI summary panel between E and the others to see the "AI tint" effect cleanly.

Section 3 · M4 Forest-Teal Bridge — v2 (darker dark mode) 1 card · contrast fix only

You flagged the original M4's dark mode as having a "dim white" — the body text wasn't popping against the surface, especially non-bold copy. The diagnosis: surface (#0E2926) and surface-raised (#1A4441) were too close in luminance to the cream text (#E8E6E0). v2 fixes it: surfaces drop to deep teal-forest near-black (#07201C and #102F2B), body text brightens to #F0EDE4, and muted text lifts from #8A958A → #A5B0A5 so non-bold paragraphs read cleanly. Light mode is byte-for-byte unchanged.
