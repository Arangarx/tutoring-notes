Mynk — typography systems
5 candidates against the brand brief · 2026-05-19

Five typography systems, same content, same color base.

Color is held constant at Concept 3 (the family-favorite palette) so this page is purely about type. Each card shows: the wordmark at scale, a heading + sub, body copy, a UI strip with a button + AI-tag + timestamp, a student-name list (proper-noun handling matters), and an AI-generated session-summary block.

Brand brief filter (already locked): warm, competent, calm, durable. Not cute, juvenile, gamified, AI-bro, or enterprise-cold. Every system below survives that filter; each one fails differently if pushed too far.

What to look for: Does "Mynk" feel like a real product name in this font? Does the AI-content block read as quotable (literary, considered) or as output (functional, plain)? Does the student-name list feel like a friend list or a database row?
