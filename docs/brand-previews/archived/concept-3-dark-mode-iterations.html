Mynk — Concept 3 dark mode iterations
brand-walkthrough phase 2 · daughter's note: teal too dominant in dark

Three iterations on the dark mode.

Original Concept 3 is in column 1 for reference. The three iterations all try to solve the same root issue (your daughter's "teal looks off with the other colors in dark mode" + your "too dominant in dark mode") but pull different design levers. Light mode is preserved across all four since the family already liked it; the toggle lets you sanity-check that each variant still has a coherent light state, but the comparison happens in dark.

Pay attention to: (1) does the coral accent now feel distinct, or does it still fight the background? (2) does the dark mode feel like Mynk, or like "any well-designed dark theme"? (3) which version would your daughter say "okay yes, this one"?
