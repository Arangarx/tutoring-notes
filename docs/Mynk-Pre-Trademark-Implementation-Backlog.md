# Mynk — Pre-Trademark Implementation Backlog

**Goal:** Use "Mynk" as if it is the product name, gather real-world signal, and avoid expensive legal commitment until usage + pilot confidence increase.

---

## P0 — Immediate brand wiring

- [ ] Replace temporary app name with "Mynk" everywhere:
  - App title
  - Browser title
  - Login screen
  - Loading screen
  - Footer
  - Metadata
  - Package / app identifiers where safe
  - Favicon placeholder
- [ ] Reserve digital identity:
  - `mynkapp.com` (if available)
  - `usemynk.com` (if available)
  - `mynkhq.com` (if available)
  - Discord
  - GitHub org
  - X / Twitter
  - Instagram
  - TikTok
  - YouTube
  - Reddit
  - Gmail / email aliases
- [ ] Create temporary logo placeholder:
  - Simple wordmark only
  - No major logo investment yet
  - Black/white works on light and dark backgrounds
- [ ] Add favicon / app icon placeholder:
  - Simple `M` / mink silhouette
  - No polish required

---

## P1 — Brand validation

- [ ] Ask 5–10 people cold:
  - **Q1:** How would you pronounce "Mynk"?
  - **Q2:** What do you think this product is?
  - **Q3:** Does it feel:
    - A) Childish
    - B) Professional
    - C) Productivity
    - D) Education
    - E) Finance
    - F) Something else
- [ ] Ask tutors specifically:
  - Would you install something called Mynk?
  - Does it sound trustworthy enough for tutoring?
- [ ] Track reactions:
  - Confusion count
  - Pronunciation count
  - Sentiment count

## P2 — Product-alignment validation

- [ ] Confirm Sarah + future pilots care most about:
  - Automatic notes
  - Session organization
  - Reduced admin burden
  - Transcripts
  - Session history
- [ ] Verify brand doesn't conflict with user perception:
  - **Yes:** "Mynk helps me teach"
  - **No:** "Mynk teaches students"

## P4 — Metrics before filing

**Do not file trademark until:**

- [ ] 5+ pilot users
- [ ] Name still feels right after 1–3 months
- [ ] Real usage exists
- [ ] No practical confusion discovered
- [ ] Product direction stable

**Then:**

- [ ] Perform broader clearance search
- [ ] Evaluate Intent-to-Use vs. direct filing
